
Anwenderhinweise:

Während das Spiel läuft, kann der Wurm über die nachfolgende Tastenbelegung gesteuert werden.
Wenn der Wurm die Ränder des Fensters erreicht, wird das Spiel beendet.
Wenn der Wurm mit sich selbst kollidiert, wird das Spiel ebenfalls beendet.
Das Spiel endet außerdem, wenn die Taste "q" gedrückt wird.

Tastenbelegung:
    
    "Pfeiltaste AUF"      - Bewegung aufwärts
    "Pfeiltaste AB"       - Bewegung abwärts
    "Pfeiltaste LINKS"    - Bewegung nach links
    "Pfeiltaste RECHTS"   - Bewegung nach rechts

    "1" - Bewegung nach links oben
    "2" – Bewegung nach rechts oben
    "3" – Bewegung nach rechts unten
    "4" – Bewegung nach links unten

    "g" - simuliert den Verzehr von einem Futterbrocken

    "s"         - schaltet Single-Step ein
    "Leertaste" - schaltet Single-Step aus

    "q" – beendet das Spiel
